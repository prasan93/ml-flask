## Local Installation

- Install pipenv
- Install dependencies
    - `pipenv shell`
    - `pipenv install -r requirements`
- Start the app
    - `python3 main.py`

- Start the app on production
  - `export FLASK_ENV=production` or `set FLASK_ENV=production`
  - `python3 main.py`

## Configuration
- Update the settings.toml file
